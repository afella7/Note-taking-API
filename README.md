# 📒 Note-Taking API  

A simple RESTful API for managing notes, built with **Node.js, Express, TypeScript, and MongoDB**.  

## 🚀 Features  
- Create, read, and delete notes  
- MongoDB for data storage  
- TypeScript for type safety  
- Express for routing  
- Basic error handling
